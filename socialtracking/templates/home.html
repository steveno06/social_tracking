Corona Virus Impact Reduction App

The Project

This is the project created for the foundations to software engineering learning unit. The aim is to provide a platform fo users to be able to easily add entries to record the people that they have met in the past. Simply create and account and log in. Once logged in the user can add a entry/tracker. Back to the homepage, users can easily see all of the records and search for a specific ones. It was created using python, html, and css. Implementing Flask and SQLAlchemy aswell.
